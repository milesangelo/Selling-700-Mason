## Quick value tips so you don’t overpay
1. **Get 2–3 written quotes** with clear scope + warranty.
2. Ask for **line‑items** (diagnostic vs repair; materials vs labor; disposal; permits).
3. Confirm **license + insurance** before scheduling work (request a certificate).
4. For stucco sagging: paying for a **diagnostic/moisture inspection** first often prevents paying for the wrong fix.
